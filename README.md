# To-Do List CLI ✅

A simple console-based to-do list manager built in Python, as part of my Python Developer Internship task.

## What it does
- Add, view, and remove tasks from the terminal
- Saves tasks to a text file (`tasks.txt`) so they persist even after you close the app

## How to run
```bash
python todo.py
```

## Tools used
- Python
- VS Code

## Sample run
```
=== To-Do List ===
1. View tasks
2. Add task
3. Remove task
4. Exit
Choose an option: 2
Enter new task: Finish internship task
Task added.
```

## Key concepts used
File handling, lists, string manipulation
